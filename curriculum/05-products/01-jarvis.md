# Jarvis

The agent runtime. Jarvis spawns, configures, secures, and connects Personal Operators to the Flow economy.

## What it does

Jarvis is the base protocol that lets a Personal Operator exist. It includes:

- **Jarvis Gateway** — agent access and connectivity
- **Security / Sentinel systems** — who or what can talk to an agent
- **Jarvis Doctor** — agent liveness and health
- **Sandboxed VM** — isolated compute for agents
- **Control layer** — configuration and lifecycle management

## How it fits

Jarvis gives the agent life. Without it, there is no compatible agent in the Flow system.
