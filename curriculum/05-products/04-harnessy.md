# Harnessy

The agent reliability layer. Harnessy makes agents more testable, controllable, and trustworthy.

## What it does

Harnessy gives structure around agent action. It answers whether an agent did the task correctly, used the right context, followed instructions, failed silently, and can improve over time. Key capabilities:

- Testing agent behavior
- Evaluating task output
- Feedback loops and improvement
- Safety boundaries and failure modes
- Integration with WorkStream verification

## How it fits

Harnessy makes the agent reliable. In the value engine, agents need accountability, evaluation, and boundaries — Harnessy provides them.
